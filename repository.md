## Tutorial: Instalar o Repository Tnds

### Caminho para o repositorio : http://tnds82.tk/repo

##### Ir ao file manager no menu SYSTEM
![REPO](http://tnds82.tk/tutorials/images/repo1.JPG "REPO")

##### Clicar em Add source
![REPO](http://tnds82.tk/tutorials/images/repo2.JPG "REPO")

##### Clicar em None
![REPO](http://tnds82.tk/tutorials/images/repo3.JPG "REPO")

##### Escrever o caminho do repositorio http://tnds82.tk/repo e clicar done
![REPO](http://tnds82.tk/tutorials/images/repo4.JPG "REPO")

##### Escolher o nome que quiserem dar a source Ex: tnds82
![REPO](http://tnds82.tk/tutorials/images/repo5.JPG "REPO")

##### Voltar a trás e clicar em SYSTEM
![REPO](http://tnds82.tk/tutorials/images/repo6.JPG "REPO")

##### Clicar em Add-ons
![REPO](http://tnds82.tk/tutorials/images/repo7.JPG "REPO")

##### Escolher Install from zip file
![REPO](http://tnds82.tk/tutorials/images/repo8.JPG "REPO")

##### Escolher o nome que deram ao source
![REPO](http://tnds82.tk/tutorials/images/repo9.JPG "REPO")

##### Escolher o ficheiro repository.tnds-7.0.x.zip e clicar em ok
![REPO](http://tnds82.tk/tutorials/images/repo10.JPG "REPO")
